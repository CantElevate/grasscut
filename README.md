# Grasscut

A small 2D idle game experiment built with [Odin](https://odin-lang.org/) and [Raylib](https://www.raylib.com/).

Cut grass. Watch it grow back. Upgrade your tools. Try to turn the entire field brown.

## Current Features

- Top-down 2D grass field
- Grid-based grass simulation
- Grass can be cut within a radius
- Cut grass gradually regrows over time
- Randomized regrowth prevents the entire field from growing in a predictable sequence
- Simple pixel-based visual style

## Building

Requires:

- Odin
- Raylib

Build with:

```bash
odin build .
